# PILOTForce Video Script (3 Minutes)

## [0:00 - 0:30] THE PROBLEM

**Visual**: Screen recording of fragmented systems - Slack with 47 links, Base Camp, multiple browser tabs, confused intern face

**Narration**:
> "Day 1 as a new intern at Salesforce. You get a Slack message with 47 links, 5 different systems to set up, and no idea what order to do them in. Onboarding is bureaucracy - a multi-step process where you repeatedly wait, chase, and navigate unclear rules with no single owner. New hires spend 2-3 days navigating disconnected systems instead of contributing. And this isn't just an intern problem - it's Salesforce's FY27 #1 priority: busting bureaucracy."

**Key visuals**:
- Slack screenshots with overwhelming links
- Multiple tabs: Base Camp, OrgFarm, GitHub, 1Password
- Timeline showing "Day 1... Day 2... Day 3..." with frustrated emoji

---

## [0:30 - 1:30] THE SOLUTION

**Visual**: Architecture diagram, dashboard overview

**Narration**:
> "Meet PILOTForce - Proficiency, Integration, Learning & Onboarding Tool. Instead of a generic checklist, PILOTForce is an agentic system that works like this:
> 
> First, your manager defines requirements through a simple JSON configuration - what tools you need, what access level, which team resources matter.
> 
> Second, an AI agent generates your personalized onboarding checklist - ordered by dependency, customized to your role, with step-by-step guidance.
> 
> Third, you see everything in one dashboard - no more tab-switching. The agent monitors your progress in the background, verifies your environment setup, and surfaces blockers in real-time.
> 
> But here's what makes it different: it's built on human-agent collaboration. The agent does what it's best at - organization, verification, monitoring. Humans do what they're best at - decisions, context, approval. The agent doesn't replace judgment; it amplifies it."

**Key visuals**:
- Manager JSON config example
- Architecture flow diagram
- Dashboard overview with callouts to agent activity panel

---

## [1:30 - 2:30] THE DEMO

**Visual**: Live dashboard walkthrough

**Narration**:
> "Let me show you Sarah Chen's experience on Day 3. Her dashboard shows she's 20% complete - she's finished 1 of 5 dev environment tasks. 
> 
> [Click on active task] She's currently installing VS Code extensions. Notice the agent activity panel on the right - the agent is checking extension compatibility in the background. 
> 
> [Show terminal output] The terminal shows real-time feedback as extensions install. When it's done, Sarah clicks 'Next Step'.
> 
> [Click next] Now she's on 'Request Permissions' - but this task requires manager approval. The agent automatically flags this as 'Awaiting Approval' and notifies her manager.
> 
> [Switch to manager view] Here's what her manager sees - a simple approval request with context. Manager clicks 'Approve', and...
> 
> [Back to intern view] Sarah is instantly unblocked. She continues to the next task.
> 
> [Show AI literacy panel] And throughout this process, PILOTForce is tracking her AI literacy progress - from PL1 to PL2, with recommended next actions. She's building AI skills from day one."

**Key visuals**:
- Progress ring showing 20%
- Active task card with terminal output
- Agent activity feed with real-time updates
- Status changing from AWAITING_APPROVAL to ACTIVE
- Manager approval interface
- AI literacy progress panel

- High-resolution screen recordings (1920x1080)
- Clean browser window (no bookmarks bar, extensions visible)
